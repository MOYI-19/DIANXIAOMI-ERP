# 店小秘排单号自动填写工具

为两个速卖通店铺的「待审核」订单批量填写拣货说明序号（排单号）。

## 使用方式

### 方式一：直接运行（开发测试）

```bash
pip install -r requirements.txt
python -m playwright install chromium
python main.py
```

### 方式二：打包后使用

```bash
python build.py
```

打包后在 `dist/店小秘排单工具.exe`，双击运行。

> **要求**：电脑需安装 **Chrome** 或 **Edge** 浏览器（Windows 11 自带 Edge）。

## 如果运行报错（常见问题）

### 1. 浏览器打不开
工具会自动尝试：**Chrome → Edge → Playwright Chromium**。
如果都打不开，请在电脑上安装 Chrome 浏览器。

### 2. 店铺筛选失败 / 找不到按钮
店小秘页面改版可能导致选择器失效。解决方法：

**查看当前页面上的关键文本：**
打开店小秘「待审核」页面，按 F12 打开开发者工具 → 在 Console 面板输入以下代码：

```javascript
// 查看所有按钮文字
document.querySelectorAll('button').forEach(b => console.log(b.innerText))
```

把看到的结果告诉开发者，调整选择器即可。

### 3. 批量拣货说明弹窗里的输入框填不进去
同样按 F12，在 Elements 面板中检查输入框的 HTML 结构，把 class 名称告诉我。
